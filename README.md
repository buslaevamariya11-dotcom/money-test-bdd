# Money Test BDD

![Java CI](https://github.com/buslaevamariya11-dotcom/money-test-bdd/actions/workflows/gradle.yml/badge.svg)

## Описание проекта: 
Данный репозиторий содержит автоматизированные тесты на основе **BDD (Behavior Driven Development)** для проверки функционала денежных переводов.

* **Стек:** Java, Gradle, Cucumber/JBehave.
* **Статус сборки:** Бейджик выше показывает текущее состояние проекта в GitHub Actions.
    * `passing` — всё отлично, тесты пройдены.

## Как запустить

1. **Клонировать репозиторий:**

   git clone [https://github.com/buslaevamariya11-dotcom/money-test-bdd.git](https://github.com/buslaevamariya11-dotcom/money-test-bdd.git)
2. **Запустить тесты:**

Bash
./gradlew test
